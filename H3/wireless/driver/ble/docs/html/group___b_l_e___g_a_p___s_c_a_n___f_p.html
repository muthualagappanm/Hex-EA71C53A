Macro Definition Documentation

◆ BLE_GAP_SCAN_FP_ACCEPT_ALL

#define BLE_GAP_SCAN_FP_ACCEPT_ALL   0x00

Default setting. Accept all advertising except directed advertising packets not addressed to this device.

◆ BLE_GAP_SCAN_FP_ACCEPT_LIST

#define BLE_GAP_SCAN_FP_ACCEPT_LIST   0x01

Accept only advertisement packets from devices in the filter accept list.

◆ BLE_GAP_SCAN_FP_ACCEPT_LIST_NOT_RESOLVED_DIRECTED

#define BLE_GAP_SCAN_FP_ACCEPT_LIST_NOT_RESOLVED_DIRECTED   0x03

Accept advertising packets like BLE_GAP_SCAN_FP_ACCEPT_LIST. Directed advertising PDUs where the TargetA is a resolvable private address that cannot be resolved are also accepted. Valid only for BLE_GAP_SetExtScanningParams.

◆ BLE_GAP_SCAN_FP_NOT_RESOLVED_DIRECTED

#define BLE_GAP_SCAN_FP_NOT_RESOLVED_DIRECTED   0x02

Accept advertising packets like BLE_GAP_SCAN_FP_ACCEPT_ALL. Directed advertising PDUs where the TargetA is a resolvable private address that cannot be resolved are also accepted. Valid only for BLE_GAP_SetExtScanningParams.

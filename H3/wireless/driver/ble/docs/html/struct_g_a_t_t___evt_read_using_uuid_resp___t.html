uint8_t attrData[BLE_ATT_MAX_MTU_LEN-ATT_READ_BY_TYPE_RESP_HEADER_SIZE]

Attribute data list. Data consists of one or more handle-value pairs.

  • Format of handle-value pair: <2-byte Characteristic Value handle> <Characteristic Value Read>
  • Number of handle-value pairs = (attrDataLength/attrPairLength)
    Note that the format of the data is little endian.

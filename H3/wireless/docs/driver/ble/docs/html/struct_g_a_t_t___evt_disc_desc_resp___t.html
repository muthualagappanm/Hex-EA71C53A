◆ infoData

uint8_t infoData[BLE_ATT_MAX_MTU_LEN-ATT_FIND_INFO_RESP_HEADER_SIZE]

Descriptor Data.
Data consists of handle-value pairs in following format: <2-byte Characteristic descriptor handle> <Characteristic Descriptor UUID>
Number of handle-value pairs = (infoDataLength/4) if p_infoData is 1
Number of handle-value pairs = (infoDataLength/18) if p_infoData is 2
Note that the format of the data is little endian.

◆ infoDataFormat

uint8_t infoDataFormat

Format of the received descriptor Data.
If 1 then p_infoData has 16-bit UUIDs.
If 2 then p_infoData has 128-bit UUIDs.
Other values are invalid.

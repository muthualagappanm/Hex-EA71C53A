◆ attrData

uint8_t attrData[BLE_ATT_MAX_MTU_LEN-ATT_READ_BY_GROUP_RESP_HEADER_SIZE]

Attribute data list.
Contains Tuples in following format: <2-byte service handle> <2-byte End Group Handle><Service UUID>
Service handle is the handle of service discovered, End Group Handle is the handle of the last attribute of the service discovered
Service UUID may be 16 or 128-bit.
Multiple tuples received in a single event have the same service UUID length
Number of tuples = (attrDataLength/attrPairLength).
Note that the format of the data is little endian.
